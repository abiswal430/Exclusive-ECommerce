# 🛒 Exclusive E-Commerce Website

## 🚀 Features
- User Authentication
- Product Management (Admin)
- Cart System
- Order Management
- Invoice Generation
- Admin Dashboard

## 🛠 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## ▶️ Run Project

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
